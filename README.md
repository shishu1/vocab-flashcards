# Vocab Flashcards

Wind power vocabulary flashcards built with HTML5, CSS3, and JavaScript.

## GitHub Pages

This repository is designed for GitHub Pages. The app runs fully in the browser and stores cards, review history, and theme settings in local browser storage.

## Data

- Import Excel files with columns named `中文`, `英文`, and `备注`.
- Export JSON backups from the app before changing devices or clearing browser data.
- Import JSON backups on another browser to move the deck and progress.
